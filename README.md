# Recipe Service

**Recipe Service** – серверная часть приложения по агрегации различных рецептов для готовки.  
Приложение создано в целях дополнительной практики в свободное время.

---

## Технологии

- **FastAPI**
  
- **JWT**
- **SQLAlchemy**   
- **Alembic**
- **Docker**   
- **Async**
